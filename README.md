# SEIZIT — Next.js Website

A full Next.js 16 project with TypeScript, Tailwind CSS v4, and Framer Motion.

## Pages
- `/` — Homepage
- `/about` — About Us
- `/services` — Services
- `/pricing` — Pricing
- `/contact` — Contact

## Tech Stack
- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS** (custom SEIZIT design tokens)
- **Framer Motion** (scroll animations, hover effects, page entrances)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features
- Framer Motion scroll-triggered fade-in animations on every section
- Hero entrance animations (slide up, stagger)
- Hover micro-interactions on all buttons and cards
- Responsive layout (mobile → desktop)
- Custom Tailwind design system matching original HTML design
- Active nav link highlighting per page
- Working contact form with success state
- Shared Navbar and Footer components
